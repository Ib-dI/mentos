git init: Initialise un nouveau dépôt Git local dans le dossier courant.

git add: Prépare les fichiers modifiés en les ajoutant à la zone de transit(staging area) avant de les enregister

git commit: Enregistre les modifications locales dans l'historique du projet.

git push: Evoie les commits locaux vers un dépot distant

git branch: Permet de lister, créer ou supprimer des branches.
